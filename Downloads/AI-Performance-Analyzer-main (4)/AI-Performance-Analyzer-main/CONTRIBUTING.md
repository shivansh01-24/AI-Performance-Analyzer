# Contributing to AI Performance Analyzer

Thanks for your interest in contributing!

## How to contribute

1. Fork the repository.
2. Create a feature branch: `feature/your-feature-name`.
3. Make your changes.
4. Run tests (if available) and ensure everything passes.
5. Open a Pull Request with a clear description.

## Coding guidelines

- Use Python 3.10+.
- Follow PEP 8 style where practical.
- Write clear docstrings for public functions.
- Prefer logging over bare `print()`.

## Reporting issues

- Use GitHub Issues.
- Clearly describe the problem and steps to reproduce.
- Attach screenshots/logs where useful.
